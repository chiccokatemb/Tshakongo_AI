
# Roadmap — Tshakongo_AI

## v1.1.0
- Vision: modèles ONNX spécialisés (armes, chiens) + réglages seuils UI
- Mémoire long terme locale (base vectorielle légère)
- App Builder GUI (éditeur web + Monaco)
- Navigation: SLAM 2D léger (hector / karto) + carte live
- Domotique: découverte MQTT auto + scènes paramétrables
- Notifications mobile (Pushbullet/NTFY/MQTT)

## v1.2.0
- Profils vocaux utilisateurs (speaker ID)
- Scénarios vocaux complexes (routines)
- Export des logs & analytics UI
- Amélioration sécurité (AppArmor/Rootless Docker)

## Backlog
- APK Android native avec commandes robot + capteurs
- Multi-caméras + enregistrement événementiel
- Support ROS 2 complet (gazebo, rviz)
